# 来了么到岗助手 (OfficeGo) v1.0.0

> 混合办公时代的团队考勤利器

**来了么到岗助手** 是一款专为混合办公（Hybrid Work）团队设计的微信小程序，帮助你和你的团队轻松记录办公状态，告别繁琐的考勤表格。

## ✨ 核心功能

### 📅 智能日历
- **一键打卡**: 点击日期即可标记“**来办公室**”，简单高效。
- **请假管理**: 长按日期标记“**请假**”，状态一目了然。
- **节假日自动同步**: 
  - 自动识别法定节假日（红色“休”）。
  - **智能补班识别**: 自动标注调休补班日（灰色“班”），即使是周末也能正常打卡，不再为调休烦恼。
- **月度概况**: 底部自动统计本月请假天数及详细的公共假期安排。

### 📊 数据概览
- **实时进度**: 首页仪表盘实时展示本月到岗达标进度。
- **智能计算**: 自动扣除节假日、加上补班日，精准计算有效工作日和本月应到天数。
- **团队去向**: 首页 Banner 实时展示“今天谁在公司”，方便协同办公。

### ⚙️ 个性化设置
- **灵活目标**: 支持自定义每月到岗比例（如 60%）。
- **取整规则**: 支持多种计算规则（向上/向下/四舍五入），适配不同公司政策。

## 🚀 使用场景
- **混合办公团队**: 需要统计每月到岗天数是否达标。
- **远程协作**: 想知道今天哪些同事在公司，方便约会议或聚餐。
- **个人考勤**: 记录自己的出勤和请假情况，心中有数。
